
# Política de Privacidade - Scanner DL criador de PDF

Esta Política de Privacidade descreve como o aplicativo **Scanner DL criador de PDF** (`com.dl.scannerpdf`) lida com suas informações. 

## 1. Coleta e Uso de Informações
O **Scanner DL criador de PDF** foi desenvolvido como um aplicativo utilitário. Valorizamos a sua privacidade e, por princípio, **não coletamos, armazenamos ou enviamos dados pessoais** para servidores externos. Todo o processamento de documentos é realizado localmente no seu dispositivo.

## 2. Permissões Requeridas
Para funcionar corretamente, o aplicativo solicita as seguintes permissões:
*   **Câmera:** Utilizada exclusivamente para capturar as imagens dos documentos que você deseja digitalizar.
*   **Armazenamento (MediaStore API):** Utilizada para salvar os arquivos PDF gerados na sua pasta de "Downloads" e para permitir que você visualize ou gerencie documentos salvos anteriormente.

## 3. Serviços de Terceiros
O aplicativo utiliza serviços do Google que podem coletar informações usadas para identificar você. Abaixo estão os links para a política de privacidade dos provedores de serviços de terceiros usados pelo aplicativo:
*   [Google Play Services](https://www.google.com/policies/privacy/)
*   [ML Kit (Google Document Scanner & Text Recognition)](https://developers.google.com/ml-kit/terms)

## 4. Segurança
Como o aplicativo não armazena dados em nuvem própria, a segurança dos seus documentos depende da segurança do seu próprio dispositivo móvel. Recomendamos manter seu sistema operacional atualizado.

## 5. Processamento de Texto (OCR)
A funcionalidade de extração de texto utiliza a tecnologia Google ML Kit. O texto extraído permanece no seu dispositivo, a menos que você decida compartilhá-lo manualmente usando a função de cópia ou compartilhamento.

## 6. Alterações nesta Política de Privacidade
Podemos atualizar nossa Política de Privacidade de tempos em tempos. Recomendamos que você revise esta página periodicamente para quaisquer alterações.

Se você tiver alguma dúvida ou sugestão sobre nossa Política de Privacidade, entre em contato conosco.
